# Start: Foundation Codegen

## Agent Workflow

1. Read `requirements.md` (high-level overview + feature index)
2. Read `./PROGRESS.md` (last progress of this feature if present)
3. Identify which feature you're working on from the Feature Index
4. Navigate to `features/[feature-name]/start.md` and follow that feature's workflow
5. Read `LEARNINGS.md` (past discoveries and mistakes)
6. **ALWAYS UPDATE LEARNINGS.md** after each completed task/milestone
